Warning:

Clicking on this button will destroy the backups for ${hostname} and all snapshots of data associated with it. All this data will be destroyed permanently.

Note:

This operation will return immediately, and run in the background. Destroying a host can take a long time, possibly hours for a large set of backups. You can check the web page for this server to check the progress, first all the backups associated with this host will be removed, then the host itself will be removed.
